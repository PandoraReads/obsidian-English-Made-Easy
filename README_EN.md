# English Made Easy (EME) - Obsidian English Learning Assistant

[![Plugin Version](https://img.shields.io/github/v/release/PandoraReads/obsidian-English-Made-Easy?label=Version&color=blue)](https://github.com/PandoraReads/obsidian-English-Made-Easy/releases)
[![Obsidian Version](https://img.shields.io/badge/Obsidian-v3.4.0%2B-purple)](https://obsidian.md)

**English Made Easy (EME)** is an immersive Obsidian learning plugin designed specifically for English enthusiasts. It seamlessly combines instant word lookup, spaced repetition (FSRS) flashcard review, and video shadowing to help you build a smooth learning experience and English corpus through one-stop learning sessions, mastering content in the most efficient way possible.

## ✨ Core Features

### 1. 🔍 Dictionary Assistant
*   **Instant Word Lookup**: Press `Ctrl/Cmd + Double-click` or `drag to select` to instantly display definitions in the sidebar.
*   **Context-Aware Capture**: Automatically captures complete sentences containing the word as example sentences during lookup, eliminating fragmented memorization.
*   **Offline Dictionary Support**: Built-in Youdao/Google/Bing online services with support for loading local **MDX/MDD** professional dictionaries.
*   **Data Persistence**: One-click add to vocabulary notebook with data synchronized to both IndexedDB and local `生词本.md` file.

### 2. 🎨 Highlight Note Cards
*   **Sidebar Management**: Aggregates `<mark>` highlighted content from current document or entire library, supporting **instant navigation** to original text via card click.
*   **Multi-Dimensional Filtering**:
    *   **Color Filter**: Quick categorization by four highlight colors: yellow, pink, blue, and green.
    *   **Keyword Search**: Real-time filtering of note content via top search bar.
*   **AI-Powered Intelligence**:
    *   **AI Translation (Languages)**: One-click precise translation of highlighted sentences.
    *   **AI Research (Microscope)**: **Three-layer philosophical depth analysis** (essence extraction, conceptual perspective, practical pathway) providing thoughtful background expansion.
*   **Flexible Interaction**:
    *   **Quick Edit**: Click pencil icon to directly edit notes on cards with auto-adjusting height.
    *   **Tag System**: Support for custom tags with built-in intelligent recommendation algorithm.
    *   **Card Collection Navigation**: One-click switch to full-screen "Card Studio" mode for centralized inspiration management.

### 3. 🧠 FSRS Flashcard Review System
*   **Scientific Algorithm**: Integrated with the most advanced **FSRS (Free Spaced Repetition Scheduler)** algorithm.
*   **Stunning Visuals**: Exquisite **Glassmorphism** UI design with 3D card flip interactions.
*   **Learning Dashboard**: Built-in learning heatmap and statistics dashboard, making growth progress clearly visible.

### 4. 🎬 Video Shadowing Workshop
*   **Multi-Mode Learning**: One-click switch between **Shadowing Mode** and **Dictation Mode**.
*   **Precise Playback Control**: Specially designed control bar:
    *   **Speed Adjustment**: Flexible switching between **0.8x / 1.0x / 1.25x** speeds.
    *   **Quick Positioning**: **-5s / -10s** quick back-jump for repeated pronunciation refinement.
*   **Full Format Support**: Perfect support for YouTube and local videos with real-time subtitle synchronization and auto-scrolling during playback.

## 🚀 Quick Start

### Installation
1. **Manual Installation (Recommended):**
    *   Obtain the latest version's three main files: `main.js`, `manifest.json`, and `styles.css`.
    *   Place them in your vault folder: `.obsidian/plugins/obsidian-english-made-easy/`.
    *   Enable the plugin in Settings > Third-party plugins.
2. **Community Plugins (Coming Soon):** Search for `English Made Easy` in the Obsidian Community Plugins market and install.

### Basic Configuration
1.  **Dictionary Settings**: Select your preferred dictionary source in settings (Youdao or local MDX recommended).
2.  **Vocabulary Notebook Path**: Specify a folder for auto-generated `生词本.md`.
3.  **AI LLM API (Important)**: To use AI Translation and AI Research features, configure an AI LLM API in plugin settings. Supports mainstream providers including OpenAI, Claude, DeepSeek, etc. Please ensure your API Key is correctly added to unlock full AI functionality.
